1. Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано предварительно?

Предварительно нужно создать репозиторий на Github на своем аккаунте, не забыв создать файл Readme. Скопировать ссылку на этот репозитоорий - SSH
Скопировать на свой комп. этот репозиторий с пом. команды git clone 'ссылка SSH'.
--------------------------------------------------------------------------------------------------------------------------------------------------------
2. Как создать git репозиторий из локального каталога на своем компьютере. Как его отправить на github.com?

                 (1) Внести правки в проект (создать папки, файлы...)
	(2) git add . (проиндексировать все)
	(3) git commit -m 'какое-то описание' (поставить в очередь на commit)
	(4) git push (выгрузить на GitHub)
---------------------------------------------------------------------------------------------------------------------------------------------------------

3. Как получить на локальный компьютер изменения с github.com?

Нужно перейти в папку с репозиторием и выполнить команду git pull
--------------------------------------------------------------------------------------------------------------------------------------------------------



 4. Какую информацию нам дают команда?

git status  -  показывает состояние раб/каталога
git log  -  показывает список комитов и когда были изменения
git remote -v  -  проверяет приивязку репоз. к Github
